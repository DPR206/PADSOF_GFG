[ Campos separados con ";", listados separados con "," ]

Se guardan y leen en este orden:

> Notas:
    - [x] indica que está hecho, [ ] que no
    - Los títulos con ---- <nombre> ---- indican qué debe imprimir el toString de esa clase
    - Como algunas navegaciones requieren preexistencia de ambos elementos involucrados "finishingTouches()" asigna los
    campos que no se pudieron asignar en su momento una vez creada la clase necesaria, se ejecuta al final de Load
    - Probablemente podría hacerse en menos archivos (incluso solo uno) pero creo que con esta separación se lee mejor y
     evita switches innecesarios

------------------------------------------------------------------------------------------------------------------------

[x] Parameters:
    ---- Parameter ----
    OFFER_TIME;ORDER_TIME;VAL_COST

[x] Categories:
    ---- Category ----
    NAME;REVENUE <- Products se rellena al crear los storeProducts

[x] Reviews:
    * ---- Review ----
    {TOTAL_ID}
    ID;SCORING;COMMENT;AUTHOR <- Los autores se guardan para los "finishing touches"

[x] StoreProducts:
    ---- Product -----
    {TOTAL_ID}
    TYPE;ID;PRICE;NAME;DESC;PHOTO
    ---- StoreProduct ----
    + super;REVIEW_IDS;AVG_PUNCT;STOCK;CATEGORIES;ADDED_DATE <- Discount se rellena al crear los Discounts
    ---- Comic / Figurine / Game ----
    + super;NUM_PAGES;AUTHOR;EDITORIAL;YEAR;NUM_PLAYERS;AGE_RANGE;GAME_STYLE;BRAND;MATERIAL;DIMENSION

[x] SecondHandProducts:
    ---- Product -----
    {TOTAL_ID}
    TYPE;ID;PRICE;NAME;DESC;PHOTO
    ---- SecondHandProduct ----
    + super;VAL_DATE;AVAILABLE;PAID_VAL;STATUS

[x] Packs:
    ---- Pack ----
    {TOTAL_ID}
    ID;PRICE;PRODUCT_IDS;DATE_ADD_CART

[ ] Discounts: <- REVISION PENDIENTE
    ---- Discount ----
    {TOTAL_ID}
    TYPE;ID;START_DATE;END_DATE;PRODUCTS;OVER_WHOLE
    ---- FixedPerDisc / GiftDisc / QuantityDisc / VolumeDisc ----
    + super;PERCENTAGE;GIFT;SPENDING_THRESHOLD;NUM_PRODS;DEDUCTION

> NOTE: Para enlazar offers y exchanges con los clientes creo que es mejor hacerlo a través de los histories

[x] Offers:
    ---- Offer ----
    {TOTAL_ID}
    ID;CREATION_DATE;ORIGIN;DESTINATION;ORIGIN_PRODUCTS;DESTINATION_PRODUCTS;STATUS <- Requiere preexistencia usuarios!!

[x] Exchanges:
    ---- Exchange ----
    {TOTAL_ID}
    ID;DATE;EXCHANGED;ORIGIN;DESTINATION;ORIGIN_PRODS;DESTINATION_PRODS <- Requiere preexistencia usuarios!!

[ ] Orders:
    ---- Order ----
    {TOTAL_ID}
